# azcam-lbtguiders

## Purpose

This repository contains the *azcam-lbtguiders* *azcam* environment.  It contains the code and data files for the LBTO lbtguiders camera systems.

## Installation

Download code and install python package:

```python
git clone https://github.com/mplesser/azcam-lbtguiders
pip install -r ./azcam-lbtguiders/requirements.txt
```

The `support` subfolder contains files and folders which may be useful in `datafolder`.  See `server_lbtguiders.py` and `console_lbtguiders.py`.

For Windows systems a common root folder would be `c:\azcam\azcam-lbtguiders`.
